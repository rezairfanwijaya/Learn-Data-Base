Alias as other than columns and tables

28 October 2021

Aliases in the database are very useful later if we learn about relations between tables. These aliases can be used for column names and table names. For column names, if the alias is more than one word, you must use quotation marks. The alias in the column name command is quite simple, you have to type the command SELECT column_name AS alias_name from table_name.And alias in the name of your table simply type SELECT alias_tabel.nama_kolom as alias_kolom from nama_tabel as alias_tabel
