Create Database For The First Time Using Mysql with CMD

28 October 2021

The MYSQL database uses a different SQL language from programming languages in general. In this case, I will discuss how to create a database for the first time using CMD. First you have to open CMD by pressing the window and R keys simultaneously then enter the command mysql -uroot -p and leave the password blank. After that you type the command create database name_database. For database names, continue not to use capital letters and spaces. Congratulations you have successfully created your first database

Then how to see the list of databases that we have created? It's very easy, you just type the show databases command and all the databases you have created will be displayed.

If you feel you no longer need the database that you have created, you can delete the database by typing drop database database_name command.
